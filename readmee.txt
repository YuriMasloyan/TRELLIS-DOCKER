Способ 1: Для гиков (через терминал)
Запусти Docker Desktop и подожди полминуты, пока движок загрузится 
(в левом нижнем углу программы должна загореться зеленая полоска или статус Engine running).

Открой абсолютно любой терминал (PowerShell или командную строку — неважно, в какой папке ты находишься).
Введи короткую команду для пробуждения контейнера:

Bash
docker start trellis-container
Подожди секунд 10-15, чтобы модель успела подгрузиться из кэша жесткого диска в память видеокарты,
 и открывай браузер: http://localhost:7860.





Способ 2: Для ленивых (через мышку и интерфейс)
Это самый простой вариант, при котором консоль не нужна вообще.
Открой Docker Desktop и дождись его загрузки.
Слева в меню перейди на вкладку Containers.
Найди в списке свой trellis-container (он будет серым со статусом Exited).
Просто нажми кнопку ▶ (Start/Play) в строке с этим контейнером.
Иди в браузер по адресу http://localhost:7860.




План Б: Если контейнер случайно удалился
Если ты когда-нибудь случайно нажмешь иконку корзины в Docker Desktop напротив контейнера, не паникуй. 
Твой правильный образ (trellis-app) и скачанные 10 ГБ весов никуда не делись. 
Чтобы пересоздать контейнер, просто открой консоль и снова введи ту самую длинную команду (папка тоже не имеет значения):

Bash
docker run -d --gpus all -p 7860:7860 --name trellis-container trellis-app
Удачных и быстрых генераций!
